Proyecto Verify App

←
→

Para esta APP me tocó trabajar en mejoras de usabilidad resolviendo un diseño más amigable para el usuario, otro de los objetivos era proponer una mejor interfaz para incrementar las descargas con los usuarios iOS; ya que la app sólo estaba disponible para usuarios android, en este nuevo diseño manejé más iconografía para describir mejor las secciones, colores más contrastantes una tipografía más legible, etc.

Verify es una aplicación que te permite consultar infracciones, debes registrarte para que te envíe notificaciones cuando tienes una multa, verifican tu auto a domicilio y te da tips sobre temas relacionados con tu auto.

Proyecto Sistema Asesoria Actinver

←
→

Para la aplicación sistema de asesoría me tocó colaborar en equipo sobre un análisis en el que aplicamos la metodología de sirius, generamos tareas para detectar los puntos de dolo que obstaculizaban a los usuarios, los resultados obtenidos en la investigación nos ayudaron a resolver las mejoras aplicadas al rediseño; consiguiendo una interfaz más amigable así como un producto mejor presentable hacia los prospectos cuando los asesores de ventas les exponían la información.

Sistema de Asesoría es una aplicación web que los asesores de ventas usan como herramienta de trabajo para llevar a cabo sus actividades diarias y control sobre sus clientes y prospectos.

Proyecto spending pocket BBVA

←
→

Proyecto aplicativo spending pocket: el objetivo era resolver una propuesta que fuese fácil de administrar, que fuese más intuitiva sin dejar de lado los lineamientos que BBVA manejaba con sus demás productos.

La aplicación permite administrar y programar los gastos por orden de prioridad, monto, frecuencia, tiempo, etc.

Proyecto Burs@net Actinver

Para Proyecto Burs@net la meta era incluir toda la ayuda necesaria y que el usuario no tuviese la necesidad de estar llamando para resolver dudas, se incluyó una ayuda general para todas las secciones dónde explicaba todo con detalle, se incluyeron videos, preguntas frecuentes, una imagen descriptiva, etc.

Proyecto Refactor Modelonow V3.0

El objetivo era llegar a más público para aumentar las ventas a través de la versión web responsiva, por lo que se implementó SEO para posicionar la marca del producto, se amplió el anaquel de productos y se agregaron mejoras de usabilidad. Mi función fue maquetar las pantallas de los diseños que el equipo ux ya tenía resueltos, también me tocó colaborar con el equipo de desarrollo para implementar y darle funcionalidad a la aplicación.

Es un E-commerce de grupo modelo que vende cerveza a domicilio, una alternativa para los usuarios que no habían descargado la app android o iOS.

Proyecto sitio web Kanzlei

Para el sitio Kanzlei me tocó trabajar en el rediseño ya que su aspecto era un poco anticuado, entre las tareas estaba renovar el sitio, incluir nuevas secciones, mejorar el contenido, crear una versión responsiva y además que el contenido se pudiese actualizar.

Sitio Kanzlei es un despacho de abogados especializados en temas de derechos sobre construcción principalmente.
